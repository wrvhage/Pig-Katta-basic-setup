Build Katta from sources:
=====

+ git clone https://github.com/sgroschupf/katta.git
+ ant test


Operate Katta
=====

+ To start/stop a katta cluster:
	+ $ bin/start-all.sh
	+ $ bin/stop-all.sh
+ To interact with a katta cluster:
	+ $ bin/katta
