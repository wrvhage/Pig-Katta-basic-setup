Katta EC2

This scripts are a port of the hadoop ec2 scripts. They are only minimal customized.
Credits go to the hadoop ec2 script authors.


This collection of scripts allows you to run Katta clusters on Amazon.com's Elastic Compute Cloud (EC2) service described at:

  http://aws.amazon.com/ec2
  
To get help, type the following in a shell:
  
  bin/katta-ec2
